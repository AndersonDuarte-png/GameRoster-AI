# GameRoster AI

GameRoster AI é um portal web local para organizar personagens de jogos com muitos atributos, formações e composições de time.

A ideia do projeto surgiu da dificuldade de gerenciar vários personagens em jogos como **Fate Grand Order** e **Wuthering Waves**, onde cada personagem possui classes, níveis, atributos, habilidades e funções diferentes dentro de uma equipe.

## Objetivo

Criar uma aplicação simples, visual e organizada para cadastrar personagens a partir de imagens, utilizando IA para identificar informações automaticamente e reduzir o trabalho manual.

## Funcionalidades planejadas

* Organização por jogo ou pasta
* Upload de imagem do personagem
* Leitura automática da imagem com IA
* Identificação de nome e atributos principais
* Busca de informações adicionais sobre o personagem
* Cadastro automático no banco de dados
* Edição manual dos dados como contingência
* Exclusão de personagens cadastrados
* Visualização clara em cards

## Exemplos de dados salvos

* Nome
* Jogo
* Classe ou tipo
* Level
* Ataque
* Habilidades
* Função no time
* Observações
* Imagem do personagem

## Tecnologias previstas

* Frontend web
* Python no backend
* MySQL como banco de dados
* Integração com IA para leitura de imagem e enriquecimento dos dados
